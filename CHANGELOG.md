# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-29

### Added
- Initial release of portfolio website
- Homepage with interactive 3D particle mesh (React Three Fiber)
- Projects page with filterable grid (AI, Robotics, IoT, XR categories)
- About page with experience/education timeline
- Contact page with form and social links
- Privacy policy page
- Responsive navigation with mobile menu
- Dark mode by default with cyan/teal accent theme
- SEO optimization (metadata, sitemap, robots.txt)
- Lazy-loaded 3D canvas for performance
- shadcn/ui components integration
- TypeScript strict mode
- Unit tests for utility functions

### Technical
- Next.js 16 App Router
- Tailwind CSS v4 with custom theme tokens
- React Three Fiber for 3D graphics
- Geist font family
- Vercel Analytics integration
